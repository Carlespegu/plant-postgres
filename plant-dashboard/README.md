# Plant Dashboard

Dashboard React + Vite per visualitzar les lectures de la teva API de Plant Station.

## Arrencar

```bash
npm install
npm run dev
```

## Configuració

A la pantalla posa:
- Base URL de l'API (ex. `https://plant-postgres-1.onrender.com`)
- API Key
- `deviceId`
- límit de lectures

## Notes

Si surt `Failed to fetch`, revisa CORS al backend FastAPI.
